# handy

一个便捷的命令行工具，集合了一些常用的功能

## 安装

```bash
uv tool install .
```

安装后可在任意路径使用 `handy` 命令。原理：uv 创建独立虚拟环境安装项目及依赖，并将 `[project.scripts]` 定义的入口链接到 `~/.local/bin`。

### 卸载

```bash
uv tool uninstall handy
```

### 注意事项

修改源码后重新安装时，若版本未生效可先清除缓存：

```bash
uv cache clean && uv tool install . --force
```

## 配置

`~/.config/handy/config.json`，字段说明：

| 字段               | 类型   | 说明                      |
| ------------------ | ------ | ------------------------- |
| `private_key`      | string | PEM 格式的 RSA 私钥字符串 |
| `private_key_file` | string | 密钥文件的绝对路径        |

密钥读取优先级：`--key` > `--key-file` > `config.private_key` > `config.private_key_file`

## 命令

### decrypt

RSA PKCS1_v1_5 分段解密。将经过 double base64 编码的密文解密为原文。

| 参数        | 必填 | 说明                                     |
| ----------- | ---- | ---------------------------------------- |
| `encrypted` | 是   | 经过 double base64 编码的 RSA 密文字符串 |

| 选项                | 说明                                    |
| ------------------- | --------------------------------------- |
| `--key <string>`    | 直接传入 PEM 格式的私钥，优先于配置文件 |
| `--key-file <path>` | 指定私钥文件路径，优先于配置文件        |

### time

解析时间戳或日期字符串，输出多种格式。不带参数或传 `now` 时输出当前时间。

| 参数    | 必填 | 说明                                                           |
| ------- | ---- | -------------------------------------------------------------- |
| `value` | 否   | Unix 时间戳（秒/毫秒自动识别）、`YYYY-MM-DD HH:MM:SS` 或 `now` |

| 选项    | 说明                            |
| ------- | ------------------------------- |
| `--iso` | 显示 ISO8601 格式（T 行带时区） |

输出内容：

| 行      | 说明                                                |
| ------- | --------------------------------------------------- |
| `ts`    | Unix 时间戳，整数秒和带毫秒精度的值（有则展示）     |
| `space` | 日期时间，空格分割（本地时区）                      |
| `T`     | 日期时间，T 分割（本地时区），需 `--iso` 参数才展示 |

### version

输出 handy 的版本号，读取自 `pyproject.toml` 的 `version` 字段。无参数。

## 开发指南

安装依赖（含开发依赖）：

```bash
uv sync
```

运行测试（覆盖率默认自动显示）：

```bash
uv run pytest
```
